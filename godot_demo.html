Crabs Godot Demo

As a final component of the Crabs framework, a Godot-based demo was created to showcase the systems in a game environment.

Overview

  • Genre: Top-down 2D interactive story
  • Focus: Branching dialogue and quest progression
  • Combat: A card game called PERSI based loosely on the rules of Mau-Mau

Core Features

  • Demonstrates full integration of Dialogue System
  • Showcases dynamic Quest System with tasks and conditions
  • Utilizes Save System to load and store player progress
  • Has multiple endings
  • Interactive card-based combat via the PERSI system

PERSI Card Game

The demo includes card battles based on a made-up game called PERSI. While inspired by Mau-Mau, it introduces its own rules and pacing.
