Detailed Description

Contains definition for Save class that is used to manage single save slot.

This header defines the Save class and related structures for managing individual save slots in the Crabs save system. It provides functionality for creating, loading, validating, and managing metadata for save files, as well as utility functions for directory management.

The SaveInfo struct holds metadata for each save slot, such as its unique identifier, display name, and creation/modification date. The Save class encapsulates operations on a save slot, including file path management, serialization/deserialization, and validation.

Dependencies:

  • <filesystem> For file and directory operations
  • <memory> For smart pointers
  • <unordered_map> For internal data structures
  • "serializer.hpp" and "deserializer.hpp" For save/load operations as json objects
  • "constants.hpp" For save file extension and other constants
  • import_export.hpp For symbol import/export macros (DLL/shared library support).
