Detailed Description

Defines the Dialogue class for managing dialogue instances within the CRABS Dialogue System.

This file contains the declaration of the Dialogue class, which represents a single dialogue instance managed by a DialogueStack. It provides asynchronous initialization, access to the underlying DialogueInterpreter, and state management for initialization. The Dialogue class is a core component of the CRABS Dialogue System, facilitating dialogue handling and interpretation.
