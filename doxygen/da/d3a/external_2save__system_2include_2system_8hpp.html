Detailed Description

Contains definition for SaveSystem class that is used for managing multiple saves and trait creation/loading.

This header defines the SaveSystem class, which provides an interface for creating, retrieving, updating, and deleting save files on disk. It manages save metadata and handles save directories.

The file also defines internal structures for associating save metadata with filesystem paths.

Dependencies:

  • <string>: For handling save names and paths as strings.
  • <vector>: Used to store and return lists of save metadata.
  • <optional>: For representing optional return values (e.g., when parsing save info).
  • <filesystem>: For interacting with the file system (directories, paths, file times).
  • <stdexcept>: For throwing exceptions on errors (e.g., invalid operations).
  • <map>: For caching save metadata keyed by save ID.
  • "import_export.hpp": For symbol import/export macros (DLL/shared library support).
  • "save.hpp": For Save and SaveInfo class definitions.
