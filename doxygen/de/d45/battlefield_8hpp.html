Detailed Description

Defines the core battlefield logic and interfaces for the PERSI card game system.

This header provides the main classes and enums for managing a card battle, including actors, moves, logging, and asynchronous/synchronous battle execution. It includes the following key components:

  • BattleLogEntryEnum: Enumerates possible battle events.
  • PlayerEnum: Identifies players in the battle.
  • BattleLog: Represents a single battle event log entry.
  • IBattleLogger: Interface for logging battle events.
  • BattleField: Manages actors, card field, turn order, and battle flow.
