◆ runBattleSerial()

void BattleField::runBattleSerial ( std::shared_ptr< IBattleLogger > log)

Runs the battle synchronously.

Parameters
logShared pointer to the logger instance.

Starts the battle with player 1 as first and runs a game loop until battle ends. Battle loop:

  1. ->Get current Actor -> actor and notify him on the start of the new turn using actor.newTurn.
  2. ->Run Inner loop that allows player to do moves
  3. -->Run actor.playNext function and get current move
  4. --> Check if move is valid and notify actor with actor.okCard
  5. --> If there was no move on moveNext, break the inner loop
  6. -> Apply damage and if health is not 0 change player. Else end the game
