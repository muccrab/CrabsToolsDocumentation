Detailed Description

Maintains parsing state including iterator position, stack, and parsed entries.

Warning
ENVIROMENT DOES NOT HOLD ITS VALUES! IF VALUES ARE DESTROYED, BAD ACCESS CAN RESULT!

Provides methods to read characters, move the iterator, manage parsing stack, and store parsed JSON tokens for deserialization stack automata
