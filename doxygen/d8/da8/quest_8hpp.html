Detailed Description

Contains definition for the Quest class that is used for managing a single quest.

This class provides the main interface for creating, managing, and interacting with quest parts, handling quest state updates, and integrating with quest strategies and callbacks.

Dependencies:

  • <any>: For passing arbitrary data to quest updates.
  • "import_export.hpp": For symbol import/export macros (DLL/shared library support).
  • "quest_part.hpp": Defines QuestPart and related types.
  • "constants.hpp": Contains constants for quest part groups and other settings.
  • "quest_strategy.hpp": Interface for quest processing strategies.
  • "quest_callback.hpp": Interface for quest event callbacks.
  • "aditions/include/multi_indexer.hpp": Provides MultiIndexer and MultiIndexHandle for managing quest parts.
