Detailed Description

Dialogue Interpreter for CRABS Dialogue System.

This file defines the core classes, structures, and enums for the CRABS Dialogue System interpreter. It provides mechanisms for parsing, executing, and managing dialogue scripts, including command handling, stack management, triggers, and input processing. The interpreter supports branching dialogue, localization, and event-driven interactions.
