Detailed Description

Defines interfaces and implementations for actors in the card battle system.

This file contains the definition of the IActor interface, which represents a player or AI controlling a deck of cards in battle, and the RandomActor class, which implements an AI that plays moves randomly. Actors manage health, interact with their deck, and decide moves based on the battlefield state.
