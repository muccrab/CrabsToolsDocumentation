Detailed Description

Contains definition of Serializer interface used to serialize complex data structures.

This header defines the core interfaces and classes for the crabs serialization framework, including the base serializer interface, value/object accessors, and a JSON serializer implementation. It provides mechanisms for serializing and deserializing objects, values, and arrays, as well as supporting visitor strategies for processing serialized data.

Features:

  • Type-safe value and object serialization.
  • Support for scalar values, objects, arrays of values, and arrays of objects.
  • Visitor pattern for processing serialized data.
  • JSON serialization implementation.

Usage:

  • Derive from ISerializer to implement custom serialization logic.
  • Use JSONSerializer for JSON output.
  • Use accessor methods to retrieve serialized data by name.

Dependencies:

  • <string>: Provides std::string for string manipulation.
  • <vector>: Provides std::vector for dynamic arrays.
  • <unordered_map>: Provides std::unordered_map for fast key-value lookups.
  • <algorithm>: Provides standard algorithms (e.g., std::find).
  • <ostream>: Provides std::ostream for output streaming.
  • <memory>: Provides std::shared_ptr for memory management.
  • "import_export.hpp": For symbol import/export macros (DLL/shared library support).
  • "exceptions.hpp": Defines exceptions used by the serializer framework.
