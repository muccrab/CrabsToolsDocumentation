Detailed Description

Factory functions for creating various types of Card objects in the CRABS card system.

This header provides inline functions to create different card types, such as normal cards, multiplier cards, and same-rank cards, by attaching appropriate components to Card instances. It also includes an automatic card type selector based on card rank.
