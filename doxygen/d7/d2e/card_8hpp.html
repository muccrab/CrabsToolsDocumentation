Detailed Description

Defines the Card class and related components for the CRABS card system.

This file contains the declarations for the Card class, its state and suit enums, and the component system for extending card behavior in the CRABS card game PERSI. The component system allows cards to have modular logic such as damage multipliers, placement restrictions, and custom behaviors.

Main features:

  • Card representation with rank, suit, and state.
  • Component-based extension via ICardCMP and derived classes.
  • Utility methods for card manipulation and querying.
