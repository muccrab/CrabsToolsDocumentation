Detailed Description

Provides grouped indexing of objects using multi-index handles for flexible group-based access.

Author
Daniel Durica
Version
0.1
Date
2025

This header defines the MultiIndexer, MultiIndexHandle, and MultiIndexerGroup templates that allow grouping of indexed objects in named sets. The main indexer (MultiIndexer) creates objects and tracks them through handles (MultiIndexHandle). Each handle can belong to multiple groups, and groups can be iterated using MultiIndexerGroup.

  • MultiIndexer<TType> manages indexed objects and group containers.
  • MultiIndexHandle<TType> provides a move-only reference to an object and tracks its group membership.
  • MultiIndexerGroup<TType> allows iteration over objects in a named group.
