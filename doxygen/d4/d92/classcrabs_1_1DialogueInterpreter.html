Detailed Description

Core interpreter that runs compiled dialogue scripts.

Handles command execution, stack management, local and global triggers, and input processing.

std::string DialogueInterpreter::processArgument ( const std::string & argument,
StackEnviroment & env,
handle_t & instance_handle )
private

Processes Argument based on specific rules.

Parameters
argumentArgument passed to the command.
envCurrent stack environment.
instance_handleHandle to the interpreter instance.
Exceptions
std::runtime_error- Incorrect parameter

Argument are processed based on these rules (preceding rules take the priority)

  1. ans -> returns value from ans register
  2. "string_constant" - constant string value
  3. param:{number} -> Branch parameter taken from the stack
  4. Variable in variable array
  5. Translates the argument with translateArgument
