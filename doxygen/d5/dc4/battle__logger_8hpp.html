Detailed Description

Godot wrapper for the BattleLogger class interfacing with the crabs::IBattleLogger.

This class bridges the native crabs::BattleLogger functionality with Godot's scripting environment. It captures battle log entries and exposes them as Godot-friendly Array objects for use in GDScript.

Provides methods to log various battle events such as battle start, battle end, turn endings, and card actions. Uses a crabs::Switcher to cache and alternate between log entries for efficient retrieval.
