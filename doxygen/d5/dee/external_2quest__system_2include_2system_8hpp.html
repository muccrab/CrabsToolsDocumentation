Detailed Description

Contains main definition for the crabs quest system.

This file defines the QuestSystem class, which is responsible for managing the registration, creation, activation, and lifecycle of quests within the crabs quest system. It provides interfaces for registering quest files, creating quests from files or serialized data, updating the quest system, and processing quests using custom strategies.

The QuestSystem maintains internal mappings for quest files and quest handles, and utilizes a MultiIndexer container to manage quest instances. It also supports callback mechanisms for quest activation and completion events.

Dependencies:

  • "import_export.hpp": For symbol import/export macros (DLL/shared library support).
  • "quest.hpp": Defines the Quest class and related quest logic.
  • "quest_part.hpp": Contains definitions for quest parts (subtasks or stages).
  • "quest_part_condition.hpp": Provides condition logic for quest parts.
  • "constants.hpp": Contains constant values (e.g., group names) used throughout the system.
  • "task.hpp": Defines the Task class, representing discrete quest objectives.
  • "quest_factory.hpp": Factory for creating quests from files or serialized data.
  • "quest_strategy.hpp": Defines strategies for processing quests (e.g., updating, filtering).
