Detailed Description

Contains definition of JSONDeserializer static object used to deserialize JSON objects to the ISerializer interface.

This header defines the JSONDeserializer class and supporting types for deserializing JSON strings into ISerializer objects. It includes parsing state management, tokenization, and stack-based parsing logic.

Dependencies:

  • "import_export.hpp": For symbol import/export macros (DLL/shared library support).
  • "serializer.hpp": Declares the ISerializer interface and related serialization logic.
