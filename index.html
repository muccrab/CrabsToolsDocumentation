Statement

Hi, my name is Daniel ÄŽurica, and I am the author of the Crabs C++ Game Development Tools. These tools provide ways to create, manage, and develop various mechanics for your engine of choice. Well, at least that's the goal. Currently, the tools are extended to the Godot Engine, but there are plans to port them to other engines as well. They are built to be extensible and do not rely on other libraries or frameworks, making them one of a kind.

What Are the Crabs Game Development Tools?

The Crabs Game Development Tools currently consist of:

  • Dialogue System
  • Quest System
  • Save System
  • Serializer
  • Addition Library

So far, these tools have the following wrapper:

  • Crabs Godot Wrapper

There is one special system that is used for card battles in demo game. This System isn't a tool for creating games, but just a fun card game simulator.
